# weather-widget

Виджет для отображения погоды.

Демо - https://serene-ardinghelli-69428f.netlify.app/

Чтобы использовать виджет на вашем сайте, добавьте следующий код:
```
<weather-widget />
<script type="text/javascript" src="{URL to the app}"></script>
```
